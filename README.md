# DiarioIA 📰🤖

Agregador automático de noticias sobre inteligencia artificial en español.
Se actualiza cada 24 horas de forma automática.

## Temas cubiertos

- Inteligencia Artificial
- Programación y Vibe Coding
- Automatización en el trabajo
- Educación tecnológica

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- GitHub Actions (cron job 24h)
- GitHub (almacenamiento de noticias en JSON)
- Vercel (deploy)

## Desarrollo local

### Requisitos

- Node.js 18 o superior
- npm

### Instalación

```bash
npm install
npm run dev
```

El sitio estará disponible en `http://localhost:3000`
